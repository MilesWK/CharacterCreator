# <div align=center>Characterizer (Name TBD)</div>

## Short Description:
This takes a description of a character, feeds it into AI, and sends you a picture of that character.

## Long Description:
My sister and her friends are making a book, and have been using AI to make images of there characters for the book. However, they aren't allowed full AI access, so I am making this program that allows them to only make an image of a character.
